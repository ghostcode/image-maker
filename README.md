# 色彩工坊 - 图片生成器

一个功能丰富、界面精美的单文件网页应用，用于生成纯色、渐变及几何图案图片。无需后端服务，直接在浏览器中运行，支持高清导出。

## 功能特性

### 五种生成模式

| 模式 | 描述 |
|------|------|
| **纯色** | 生成任意纯色图片，支持取色器、HEX 输入及预设快选 |
| **线性渐变** | 多色标渐变，支持 0-360° 角度调节 |
| **径向渐变** | 从中心向外的渐变，可调中心位置与半径 |
| **条纹图案** | 经典条纹背景，可调宽度与倾斜角度 |
| **网格图案** | 棋盘格/方格图案，可调格子大小 |

### 核心能力

- **多色标支持**：渐变模式下可动态添加/删除颜色节点
- **精选配色方案**：内置 8 种专业配色，一键应用到当前模式
- **自定义尺寸**：支持 1920×1080、1080×1920、1080×1080、2560×1440 预设，或手动输入任意尺寸（100-8000px）
- **高清导出**：一键下载 PNG 格式图片，保留完整分辨率

### 界面设计

- 深色主题，突显颜色展示效果
- 流动光晕背景与网格纹理，增加视觉层次
- 平滑过渡动画与微交互反馈
- 完全响应式布局，适配桌面与移动设备

## 快速开始

### 使用方法

1. 将代码保存为 `index.html` 文件
2. 使用现代浏览器直接打开
3. 在左侧控制面板选择生成模式
4. 调整参数，实时预览效果
5. 设置输出尺寸
6. 点击「下载图片」保存

### 系统要求

- Chrome / Firefox / Safari / Edge 最新版本
- 需启用 JavaScript

## 操作指南

### 纯色模式
- 点击颜色块打开取色器
- 或直接输入 HEX 颜色值（如 `#00d4aa`）
- 点击预设颜色快速选择

### 线性渐变
- 拖动滑块调整渐变角度
- 点击「添加色标」增加渐变颜色
- 输入位置百分比控制颜色分布
- 点击红色 × 移除多余色标

### 径向渐变
- 调整半径控制渐变范围
- 调整中心 X/Y 控制渐变起点位置
- 同样支持多色标管理

### 条纹 / 网格
- 调整宽度/格子大小
- 分别选择两种颜色
- 条纹模式支持角度旋转

## 技术栈

- **HTML5 Canvas** - 图像渲染核心
- **CSS3** - 变量、Grid、Flexbox、动画
- **Vanilla JavaScript** - 无框架依赖
- **Tailwind CSS (CDN)** - 快速布局
- **Google Fonts** - Space Grotesk / JetBrains Mono
